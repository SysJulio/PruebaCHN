# CHN Prestamos Bancarios

Sistema web para gestionar clientes, solicitudes de prestamos, aprobaciones, rechazos, prestamos aprobados y pagos en efectivo.

### Paso 1: Obtener el proyecto

Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Entrar a la carpeta:

```bash
cd NOMBRE_DEL_REPOSITORIO
```

Si el proyecto ya fue descargado manualmente, abrir la terminal en la carpeta raiz del proyecto.


## Paso 2: Ejecutar con Docker Compose


```bash
docker compose up --build
```

Este proceso puede tardar la primera vez porque descarga imagenes y dependencias.


### Paso 3: Abrir la aplicacion

Cuando los servicios esten arriba, abrir:

```text
http://localhost:5173

```
